# c1-08 幻觉初识：找出埋下的 2 处幻觉       ⏱ 25min | 能力域 9 | 判题 CI

## 背景
AI 会一本正经地调用**不存在的 API** —— 这是最典型的幻觉形态。本关的代码"看起来很对、跑起来必炸"，其中恰好埋了 2 处，另有 1 处是故意放的正常写法（别误伤）。

## 任务
1. 只读不跑，审查 `starter/report_gen.py`，找出 2 处幻觉
2. 写 `findings.md`：每处一行，格式 `行号 | 问题代码片段 | 为什么是幻觉 | 正确写法`
3. 最后运行 `python3 starter/report_gen.py` 验证你的判断（观察报错）

## 判题
- 🟡 CI：findings.md 精确命中两处（涉及的标识符都被指出），且未把正常写法误报为幻觉
- 🟢 教练抽查：说出"遇到疑似幻觉 API 的三步验证法"（查官方文档 / 最小复现 / 换表述再问）
