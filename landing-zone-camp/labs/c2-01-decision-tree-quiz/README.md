# c2-01 Spec 决策树 20 题速判              ⏱ 20min | 能力域 3 | 判题 CI

## 背景
Spec 分级（L0 免 / L1 快速 / L2 标准 / L3 完整）是团队统一"什么改动写多重的 Spec"的共同语言 —— 对照 [handbook/spec-levels.md](../../handbook/spec-levels.md) 的分级规则与决策树逐题判定。

## 任务
判定 `starter/questions.md` 的 20 个场景，答案写入 `answers.txt`，每行 `N:L0`~`N:L3`。

## 判题
- 🟡 CI：20 题全对（哈希比对，可反复提交）
- 🟢 教练抽查：任选 3 题走一遍 [handbook/spec-levels.md](../../handbook/spec-levels.md) 的决策树分叉，讲清"为什么不是相邻级别"

## 提交方式
分支 `camp/c2-01/<工号>`
