# c1-01 部门默认工具安装与配置校验        ⏱ 20min | 能力域 1 | 判题 CI

## 背景（为什么练这个）
统一规范之前，先保证每个人的默认工具装好、配好、登记在册。
本关按你所在部门的默认工具（见平台「新手引导」页的部门默认工具区块：如 A 部门 Claude Code、B 部门 Qoder）完成环境自检。

## 任务
1. 安装并登录部门默认工具，确认能正常对话与生成代码
2. 复制 `starter/toolcheck.md` 到本目录为 `toolcheck.md`，如实填写全部字段（工具名、版本、`--version` 输出、配置文件路径、一次生成的截图说明）
3. 在平台「新手引导」页核对你部门的默认工具；有卡点写进 toolcheck.md 的「卡点」一节

## 判题
- 🟡 CI：toolcheck.md 存在且五个字段均非空
- 🟢 教练抽查：卡点描述真实具体，可直接进部门卡点表

## 你会用到
平台「新手引导」页的部门默认工具区块 · [handbook/00-index.md](../../handbook/00-index.md) 的「部门默认工具」条目

分支 `camp/c1-01/<工号>`，推送后 CI 自动判题
