# c1-05 L0/L1 分级判断 10 题               ⏱ 15min | 能力域 3 | 判题 CI

## 背景
"不是所有改动都值得写一份完整的 Spec"。分级判断准，才不会把规范变成负担 —— 这是分级判定的入门版（只判 L0 免 Spec / L1 快速 Spec）。分级标准见 [handbook/spec-levels.md](../../handbook/spec-levels.md)。

## 任务
逐题判定 `starter/questions.md` 中 10 个改动场景该走 L0 还是 L1，
把答案写入本目录 `answers.txt`，每行一题，格式 `N:L0` 或 `N:L1`。

## 判题
- 🟡 CI：10 题全对（哈希比对，改到全对为止 —— freeCodeCamp 式即时反馈）
- 🟢 教练抽查：任选 2 题说出判定依据（对照 [handbook/spec-levels.md](../../handbook/spec-levels.md) 的决策树哪个分叉）
