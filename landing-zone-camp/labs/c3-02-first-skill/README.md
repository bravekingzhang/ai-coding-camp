## 背景
本关的 🟢 不由脚本给，由**另一位跨组学员按你的 Skill 独立完成任务的成功率**决定 —— 资产好不好，唯一标准是别人用得起来。

## 任务

把它提炼为 `skills/pagination-deep-scroll/SKILL.md`：
frontmatter（name/description/version/owner/tags 五字段）+ 触发时机 / 步骤 / 反例 / 边界 / 维护信息 五节。
步骤必须可执行到"别人不问你也能做完"。

## 判题
