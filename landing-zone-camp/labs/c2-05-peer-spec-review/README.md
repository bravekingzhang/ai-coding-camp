# c2-05 Review Checklist 跨组互评          ⏱ 40min | 能力域 4 | 判题 PEER（平台闭环）

## 背景
评审别人，是把标准长进自己身体最快的方式。本关不判文件，判**行为**：完成平台指派给你的真实跨组评审。

## 任务
1. 打开平台「我的评审」，认领 1 个待评任务（来自别组 c2-02 的 L2 Spec）
2. 逐项过 Checklist（可测性/异常流/非功能/回滚），写 ≥2 条具体改进建议后提交
3. 在本目录写 `reflection.md`：本次评审你抓到的最有价值问题 + 一条你自己 Spec 也会犯的毛病

## 判题
- 🟡 平台确认你已完成 ≥1 次评审 + reflection.md 两节齐全（CI 只查后者）
- 🟢 教练对你的评审意见质量抽评 ≥8 分

## 提交方式
分支 `camp/c2-05/<工号>`（评审在平台内完成）
