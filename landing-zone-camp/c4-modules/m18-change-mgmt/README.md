# M18 变革管理（C4 · 工程教练）            ⏱ 2h

## 学习目标
把"champion 个人魅力驱动"升级为"机制驱动"，正面处理两极分化。

## 内容
1. 两极分化画像与对策：观望者（结对 + 从真实痛点小任务切入，不讲愿景讲省时间）/ 迷信者（前置幻觉与评估模块，让他亲手抓到 AI 的错）
2. 抵触信号识别：连续缺席分享会、test-out 全跳但项目拖延 —— 风险名单的人工解读
3. 异地带教：南京/上海的仪式线上化、材料自含化检查单
4. 失败案例复盘（讨论课）：为什么"行政命令统一工具"会反弹，"先统规范"为什么能软着陆

## 实操作业（计入 P3）
认领 2 名风险名单学员做转化陪跑 2 周，记录干预动作与结果（成败都算完成，复盘质量计分）。

## 通过标准
🟡 陪跑记录完整；🟢 至少 1 名学员回到正常进度
