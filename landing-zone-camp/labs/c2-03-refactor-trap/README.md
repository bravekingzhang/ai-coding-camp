# c2-03 「禁止越级」陷阱题                 ⏱ 20min | 能力域 3 | 判题 CI

## 背景
规则："你说是纯重构但实际改了行为 → 退回，补 L1 Spec。" 本关给你一个自称 L0 纯重构的 diff —— 它撒谎了。

## 任务
审查 `starter/refactor.diff`（提交者声明：纯重构，L0 免 Spec）。写 `verdict.md`：
1. 「判定」：该走 L0 还是 L1？
2. 「行为变更点」：指出 diff 中偷偷改变的外部行为（引用具体代码）
3. 「应补的 Quick Spec」：为该行为变更补 3 行以内的 WHEN/THEN

## 判题
- 🟡 CI：判定为 L1 + 指出排序方向变化 + 补了 WHEN/THEN
- 🟢 教练抽查：说明 CI 里如何自动拦截此类"伪重构"（对比测试快照/行为回归）
