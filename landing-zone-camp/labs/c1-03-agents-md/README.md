# c1-03 给项目写 AGENTS.md                 ⏱ 30min | 能力域 2 | 判题 CI

## 背景
AGENTS.md 是 AI 进入项目的"入职手册"。写得好，AI 第一次生成就贴近团队习惯；不写，每个人都在重复口头交代。

## 任务
阅读 `starter/` 下的迷你订单服务（两个文件），为它编写 `AGENTS.md`，必须覆盖六节：
项目概述 / 技术栈 / 目录结构 / 编码约定 / 禁止事项 / 常用命令。
「编码约定」至少 3 条、「禁止事项」至少 2 条，且要与 starter 代码实际情况一致（如：金额用分存储）。

## 判题
- 🟡 CI：六节齐全、总长 ≥ 800 字节
- 🟢 教练抽查：拿你的 AGENTS.md 喂给 AI 生成一个新接口，产物明显更贴合约定
