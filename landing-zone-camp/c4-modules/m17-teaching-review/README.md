# M17 教学与评审（C4 · 工程教练）          ⏱ 3h

## 学习目标
成为可靠的"人肉金标准"：判分一致、反馈有效、能带模块。

## 内容
1. rubric 校准练习：全体教练背靠背评同 5 份 Spec，比对分差，分差 >2 分的条目现场议定口径（这是每周校准会的排练）
2. LLM 预评的抽查方法：只看 20% 但要覆盖三档颜色；争议样本归档进 rubric 修订清单
3. 反馈三明治的工程版：先指认对的 → 指出最重要的一个问题（不是全部）→ 给出下一步最小动作
4. 口头答辩三问设计：怎么问出"真会/背答案"的区别

## 实操作业（计入 P3）
独立带一个 C1/C2 模块的 office hours 一周（答疑记录 ≥5 条），或主评 10 份 c2-02 的 L2 Spec。

## 通过标准
🟡 完成带教且学员匿名评价 ≥4/5；🟢 你的判分与评审团基准一致率 ≥90%
