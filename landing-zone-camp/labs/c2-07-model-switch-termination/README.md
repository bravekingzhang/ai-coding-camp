# c2-07 换个模型再审：终止条件实践          ⏱ 30min | 能力域 6 | 判题 CI

## 背景
"换个模型又能发现新问题"会导致审查无限递归 —— 多轮 AI 审查每轮都可能冒出新意见，若无终止条件就会无限递归。解法不是审到零问题，而是**约定终止条件**（见 [handbook/methods.md](../../handbook/methods.md) 的"AI 审查终止三条件"）。

## 任务
以 c2-06 的 `user_service.py`（你修复后的版本）为对象：
1. 第一轮：用你的默认模型/工具做四维审查，结果记入 `review-log.md` 的 `## 第一轮`
2. 第二轮：换模型（或换审查提示词角色），新发现记入 `## 第二轮`，并标注每条是"真问题 / 风格偏好 / 误报"
3. `## 终止判定`：按三条终止条件逐条判定是否收敛 —— ①连续一轮无新增"真问题" ②剩余条目均为风格偏好 ③高危维度（安全/正确性）为零

## 判题
- 🟡 CI：三节齐全、第二轮含分类标注、终止判定引用了全部三条条件并给出结论
- 🟢 教练抽查：口述"如果第二轮又冒出一个真问题，第三轮之后怎么办"（时间盒 + 升级人工仲裁）
