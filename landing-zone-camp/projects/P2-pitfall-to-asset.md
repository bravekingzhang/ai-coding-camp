# 认证项目 P2「从踩坑到资产」— C3 绿带毕业项目
⏱ 2 周（W8~W9）| 个人或 2 人 | 判定：SkillHub 审批 + 跨团队采纳确认

## 目标
选一个**本团队真实反复踩的坑**，工程化为：1 个 Skill + 1 条 Rule + 1 条 detection-matrix 条目。

## 必经里程碑
1. 坑的原始记录（时间线、损失、根因）
2. SKILL.md 达到 c3-02 质量标准并通过互测
3. Rule 条目提交进部门 rules 表（Rule 的定义见 [handbook/skill-guide.md](../handbook/skill-guide.md#一句话区别)）
4. detection 条目：该坑如何被 CI/扫描自动拦截
5. SkillHub 正式命名空间审批通过 → 推广给 ≥1 个其他团队 → 采纳方组长在平台确认

## 验收
- 🟡 五里程碑齐全 + 1 次已确认采纳
- 🟢 采纳 ≥2 团队，或 detection 条目实际进了门禁 v1
## 防刷条款
互相"对倒采纳"无效：采纳方必须提交真实使用证据且由其组长确认。
