# c3-03 SkillHub 全流程走通                ⏱ 30min | 能力域 11 | 判题 MANUAL（教练核）

## 背景
企业内部技能资产平台 SkillHub（什么是 SkillHub 见 [handbook/skill-guide.md](../../handbook/skill-guide.md)）的卡点从来不在搭建，在"刻意用起来"。本关把 c3-02 的 Skill 走一遍完整生命周期。

## 任务
在 SkillHub 的 `camp-1` 练习命名空间完成：上传 c3-02 的 Skill → 提交审批 → 审批通过后发布 v1.0.0 →
修改一处后发布 v1.0.1（体会 SemVer）→ 在自己工具里安装并触发一次。
把每步的界面证据（链接/截图存内网）记入 `skillhub-log.md`，五步一节。
