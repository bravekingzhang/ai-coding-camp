# M16 治理（C4 · 工程教练）                ⏱ 2.5h

## 学习目标
能主持资产周会，让 Spec、Skill、Rules 保持"活的"状态而不是文档坟场。

## 内容
1. Spec 三层防腐实操：上线即归档 archive/、PR 关联检查（门禁 v1 的可选项）、月度清理标记
2. 资产周会议程模板：新增审批 → 采纳数据回顾 → 淘汰候选（复用 c3-04 判定规则）→ rules 修订
3. Enterprise Policy 三层继承（Enterprise > Organization > User）在工具侧如何落位；哪些必须企业层锁死（如禁 YOLO 模式）
4. 判题→门禁迁移检查单：哪些 .verify 检查项可以进生产 pre-commit / CI

## 实操作业（计入 P3）
主持一次真实资产周会（W6 或 W10），会议纪要 + 至少 1 项淘汰/合并决议落地。

## 通过标准
🟡 周会开成且有决议；🟢 决议在两周内可验证落地
